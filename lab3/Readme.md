# Лабораторная работа №3
## 1. Введение
Цель работы: Закрепление теоретического материала и практическое освоение основных возможностей базовых алгоритмов растеризации отрезков и кривых.

Задачи:
* Реализация четырех алгоритмов растеризации
* Создание графического интерфейса для визуализации
* Анализ временных характеристик алгоритмов
* Обеспечение удобного интерфейса с системой координат

## 2. Реализованные алгоритмы
* Пошаговый алгоритм. (Определяет доминирующую ось (X или Y) и вычисляет координаты точек вдоль этой оси, корректируя координаты по второй оси на основе накопленной ошибки.)
* Алгоритм ЦДА. (Вычисляет приращения координат и последовательно добавляет точки с округлением до ближайших целых значений.)
* Алгоритм Брезенхема для отрезков. (Использует целочисленную арифметику и переменную ошибки для определения следующей точки без вычислений с плавающей точкой.)
* Алгоритм Брезенхема для окружности. (Строит окружность по 8 точкам симметрии, используя целочисленные вычисления для определения следующей точки.)

## 3. Сравнительный анализ алгоритмов
Для анализа производительности были проведены замеры времени выполнения каждого алгоритма на отрезке длиной 50 пикселей:

| --------------- | ---------------- | ---------------- | --------------- | ---------------- | ---------------- | ---------------- |

* Алгоритм Брезенхема для отрезков показал наилучшую производительность

## 4. Графический интерфейс. Особенности реализации
### Система координат:
* Используется дискретная сетка с настраиваемым размером ячейки
* Целочисленные координаты привязаны к узлам сетки
* Оси координат подписаны для удобства ориентации

### Элементы управления:
* Выбор алгоритма из выпадающего списка
* Настройка размера сетки
* Визуализация начальной (зеленый) и конечной (красный) точек
* Статусная строка с подсказками

## 5. Заключение
В ходе работы были успешно реализованы и протестированы четыре алгоритма растеризации. Созданное приложение позволяет наглядно изучать принципы работы алгоритмов, сравнивать их производительность и анализировать получаемые результаты.
