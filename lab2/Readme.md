# Лабораторная работа №2
Приложение для обработки изображений с графическим интерфейсом, написанное на Java. Программа позволяет загружать изображения и применять к ним различные методы улучшения контраста.

## Функциональность
### Основные возможности:
* Загрузка изображений - поддержка различных форматов через ImageIO
* Линейное контрастирование - автоматическое растяжение гистограммы для улучшения контраста
* Эквализация гистограммы - выравнивание гистограммы для оптимального распределения яркостей

### Интерфейс:
* Простой и интуитивно понятный графический интерфейс
* Отображение оригинального и обработанного изображения
* Кнопки для быстрого доступа к функциям обработки

### Используемые технологии
* Java Swing - для создания графического интерфейса
* Java AWT - для работы с изображениями и графикой
* ImageIO - для загрузки и чтения файлов изображений
* BufferedImage - для обработки пиксельных данных

### Использование
#### Пошаговая инструкция:
1. Загрузка изображения:
* Нажмите кнопку "Load Image"
* Выберите файл изображения в диалоговом окне
* Оригинальное изображение появится в левой части окна
2. Применение линейного контрастирования:
* Нажмите кнопку "Linear Contrast"
* Обработанное изображение появится в правой части окна
3. Применение эквализации гистограммы:
* Нажмите кнопку "Histogram Equalization"
* Результат обработки отобразится справа

### Особенности работы
* Программа автоматически конвертирует цветные изображения в градации серого
* Все операции применяются к текущему загруженному изображению
* Можно последовательно применять разные методы обработки

### Основные алгоритмы
1. Линейное контрастирование
* Находим минимальную (min) и максимальную (max) яркость на изображении
* "Растягиваем" диапазон яркостей на всю шкалу 0-255
* Усиливает контраст

2. Эквализация гистограммы
* Строим гистограмму распределения яркостей
* Вычисляем кумулятивную (интегральную) гистограмму
* Перераспределяем яркости так, чтобы они занимали весь диапазон равномерно
