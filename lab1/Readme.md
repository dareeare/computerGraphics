# Лабораторная работа №1
Программа ColorModelConverter представляет собой графическое приложение на Java для конвертации цветов между тремя популярными цветовыми моделями: RGB, CMYK и HLS. Приложение предоставляет интуитивно понятный интерфейс для просмотра и редактирования цветов в различных моделях с возможностью выбора цвета из палитры.

## Функциональность
### Конвертация между цветовыми моделями:
* RGB (Red, Green, Blue)
* CMYK (Cyan, Magenta, Yellow, Key/Black)
* HLS (Hue, Lightness, Saturation)

### Визуальное представление:
* Основная панель предпросмотра цвета
* Слайдеры для регулировки компонентов цвета
* Текстовые поля для точного ввода значений
* Кнопка выбора цвета из системной палитры

### Синхронизация данных:
* Автоматическое обновление всех моделей при изменении любой компоненты
* Валидация вводимых значений
* Защита от циклических обновлений

## Использование
Основные элементы управления:
### RGB панель:
* Слайдеры R, G, B (0-255)
* Текстовые поля для точного ввода
### CMYK панель:
* Слайдеры C, M, Y, K (0-100%)
* Текстовые поля для точного ввода
### HLS панель:
* Слайдер H (0-360°)
* Слайдеры L, S (0-100%)
### Панель предпросмотра:
* Отображение текущего цвета
* Кнопка "Выбрать цвет из палитры"
